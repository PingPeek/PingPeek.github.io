隐私政策

最后更新：2026 年 3 月 15 日

核心承诺：NetTweak 的所有网络流量数据仅在您的设备本地处理，不会上传至任何外部服务器。

1. 我们收集的信息

NetTweak 不收集任何个人身份信息。应用在本地存储的数据包括：

  • 网络流量统计（上行/下行字节数、实时网速）
  • 用户设置的弱网模拟参数（限速、延迟、丢包率等）
  • 流量会话记录（存储于设备本地 SQLite 数据库）

2. 我们不收集的信息

  • 您的网络请求内容或 Payload
  • 访问的网址或域名
  • 账户信息、姓名、邮箱或任何个人标识
  • 设备位置信息

3. VPN 权限说明

NetTweak 使用 Android 系统 VPN Service API 建立本地虚拟网卡（tun 接口），用于在设备内部拦截、分析并模拟网络条件。

这一机制完全在本地运行，不将任何流量转发至外部代理或服务器。VPN 权限仅用于实现弱网模拟和流量统计功能，不需要 Root 权限。

4. 数据存储与安全

所有数据存储于您设备的私有应用目录（Android 沙箱保护），其他应用无法访问。数据不会通过任何渠道离开您的设备。

5. 第三方服务

NetTweak 不集成任何第三方分析、广告或数据收集 SDK。

6. 儿童隐私

本应用面向开发者和技术用户，不面向 13 岁以下儿童，不会有意收集儿童信息。

7. 政策变更

如本政策发生重大变更，我们将通过应用更新说明予以通知，并更新本页面顶部的日期。

8. 联系我们

如有隐私相关问题，请通过 GitHub Issues 联系我们。
